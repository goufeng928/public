本文将通过 XX市XX区人民医院HRP信息系统建设项目实例 来论述 `采购管理` 在其中的作用。

## 采购管理定义

**信息系统项目管理师教材定义**:

```txt
官方教材定义混乱, 参考以下 "一句话总结"。
```

**壹句话总结**:

```txt
项目采购管理是为完成项目工作从项目团队外部购买或获得所需的产品、服务或成果的过程。
```

## 如何进行项目采购管理

**(1) 规划采购管理**:

```txt
规划采购管理:
规划采购管理需要做以下几个方面的工作:
1. 供应商管理(供应商选择指标, 评估方法等)
2. 产品采购目录管理(根据供应商所能提供的产品, 建立<<产品采购目录>>)
3. 采购需求管理(识别需求并对需求进行确认和控制)
4. 采购计划管理(采购计划是指企业管理人员对计划期内物料采购管理活动所做的预见性的安排和部署)

规划采购管理的方法:
自制或外购分析, 专家判断等

规划采购管理的输出:
采购管理计划, 采购文件等
```

**(2) 实施采购**:

```txt
实施采购:
实施采购需要做以下几个方面的工作:
1. 询价比价管理
2. 采购谈判
3. 采购合同签订

实施采购的方法:
投标人会议, 采购谈判, 专家判断等

实施采购的输出:
选定的卖方, 协议等
```

**(3) 控制采购**:

```txt
控制采购:
控制采购是管理采购关系, 监督合同执行情况, 并根据需要实施变更和采取纠正措施的过程。
在控制采购过程中, 通过对比采购合同以及卖方当前的绩效水平, 考察卖方在未来项目中执行类似工作的能力。

控制采购的技术:
采购绩效评审, 检查与审计等

控制采购的输出:
工作绩效信息, 变更请求等
```

**(4) 结束采购**:

```txt
结束采购:
结束采购是完结本次项目采购的过程。
完成每一次项目采购, 都需要结束采购过程。
它是项目收尾或者阶段收尾过程的一部分。
它把合同和相关文件归档以备将来参考, 因为项目收尾或者阶段收尾过程已核实本阶段或本项目所有工作和项目可交付物是否是可接受的。

结束采购的技术:
采购审计, 采购谈判, 记录管理系统

结束采购的输出:
结束的采购, 组织过程资产更新
```

**注意**:

采购管理一般涉及到招投标。

**中华人民共和国招投标程序**:

```txt
依据<<中华人民共和国招投标法>>, 招投标程序如下:
(1) 招标人采用公开招标的方式, 应当发布招标公告。
    招标人采用邀请招标方式的, 应当向三个以上具备承担招标项目的能力, 资信良好的特定的法人或者其它组织发出投标邀请书。
(2) 招标人根据招标项目具体情况, 可以组织潜在投标人踏勘项目现场。
(3) 投标人投标。
(4) 开标。
(5) 评标。
(6) 确定中标人。
(7) 订立合同。
```

## XX市XX区人民医院HRP信息系统建设项目(采购管理)实例

**项目背景**:

```txt
# 2023 年 9 月, 我公司通过公开招标的方式中标了 XX 市 XX 区人民医院 HRP 信息系统建设项目, 我作为乙方项目经理全程参与了该项目的建设工作。
  本项目合同金额 3,630,000.00 元, 主要建设内容包括基础平台系统、财务会计系统、全面预算管理系统、科室成本核算系统、项目成本核算系统、病种成本核算系统、固定资产管理系统、合同管理系统、人力资源管理系统、统一对账系统、智能发放系统、报销管理系统、移动应用、决策分析(BI)、接口服务的建设。
  建设工期 8 个月, 前 5 个月内完成项目实施建设, 后 2 个月上线试运行, 最终验收 1 个月。
  该项目采用 B/S 架构, 前端展现包括系统入口、总览和数据仪表盘、小程序和移动APP, 后端主要采用Java、C#等语言开发模块, 实现预算编制模块、成本控制模块、资源管理模块、财务分析模块、凭证管理模块、成本核算模块、员工信息模块、薪酬绩效模块、报表与分析模块、集成模块等。
  关键技术指标要求实现系统的用户、权限、引入、输出、日志等功能, 要求支持 EAI 集成, 要求实现 PDM 系统接口、配置、规则等, 要求实现财务会计和预算会计双重功能平行记账、支持新会计制度要求的相关会计报表的生成, 要求实现资产条码管理, 报销单自动分摊、归集等。
  该项目于 2024 年 5 月顺利完工并通过客户验收, 获得了客户的肯定。
  本文以该项目为例, 结合作者实践经验, 探讨信息化系统项目建设过程中在采购管理方面经常遇到的问题及解决方法。
  主要就如何做好采购规划, 实施采购, 控制采购, 结束采购这四个方面来进行阐述。最后总结项目中的成功经验, 以及存在的不足之处和具体改进措施。
```

**采购管理论述**:

```txt
# 鉴于该项目对公司意义重大, 公司为此组建了单独的项目团队并任命我为项目经理。
  收到委任后, 我仔细分析了项目合同、工作说明书后得出: 本项目范围广、工期紧、预算有限、采购件众多。
  怎样规划采购为采购工作提供指南和方针; 怎样实施采购为项目取得最佳的经济效益; 怎样控制采购以保障项目资源供应, 这些都是本项目需要重点考虑的问题。
  因此我认为有效的采购管理是该项目成功的关键。

# 由于采购管理在该项目中显得尤为重要, 作为项目经理我在本项目中除了对其它各管理领域恪尽职守外, 还特别对采购管理从以下几个方面进行了管理:

# 一、规划采购管理
* 作为项目经理我深深的知道科学的采购是实现项目利润最大化的源泉之一, 也是项目在质量和时间上得以保障的重要手段, 所以必须在实施采购之前对采购活动进行合理规划。
* 在本项目中, 我带领团队成员结合已分析出的需求文件、项目进度计划和成本预算等对完成项目工作所需的物资进行了详细分析。
  主要通过自制和外购分析技术分析出需要采购的项; 通过对采购项的分析, 确定出采购的方式; 通过分析进度计划中对这些采购项的需求, 确定出采购时间; 通过对这些采购项进行成本估算, 确定出采购底价。
  我们还对这些采购项在关键指标上做了必要要求, 最后经汇总和整理后制订出了采购计划。

# 二、实施采购
* 由于本项目中的配套硬件设备比较专业, 我公司并不具备自制能力, 因此公司决定采用公开招标的方式进行采购。
  因为我们具备相关招标经验且为了节约成本, 所以我们自行办理了招标事宜。
* 我们首先制订了招标文件, 在此文件中我们对采购内容、预算金额、投标方需具备的资格、投标方式、质量要求、技术标准、评标方法、开标时间等都做了详细描述。
  然后我们在政府采购网上发布了招标公告并进行了 20 天的公示。
  在收到投标文件后, 我们对投标单位进行了资格预审, 筛选出符合条件的供应商共 4 家。
* 我们在规定的开标时间和地点进行了开标, 并根据评标委员会的推荐确定了中标人。
  中标人确定后, 我们向所有投标人发布了中标结果通知, 并与中标人在中标通知书发出之后的第 5 日签订了书面合同。
  我们还依据合同条款向其支付了 30% 的预付款, 之后采购合同正式执行。

# 三、控制采购
* 控制采购是管理采购关系、监督合同执行情况, 并根据需要实施变更和纠正的过程。
  在这一个过程中, 我主要通过对比合同内容和卖方当前的绩效水平, 来考察卖方在未来项目中执行工作的能力。
* 在本项目硬件采购合同中约定, 所有配套硬件设备都必须要在签署合同后的 2 个月内到位。
  在离约定时间前的半个月, 供应商以近期受春节影响, 工厂还未开工为由, 原材料采购困难为由, 希望能延迟供货时间。
  但是我查阅工厂信息并打电话确认发现, 该供应商的主要原材料供应商均已开工。
  因此我拒绝了他们的请求, 并给该供应商负责人打电话进行了沟通。
  在随后的时间里, 我安排项目助理对该采购项进行了严密的跟踪和跟催, 最后所有设备均在合同约定时间内到场。

# 四、结束采购
* 在完成每一次采购后都需要结束采购过程。其目的是为了审计采购活动、总结经验教训并将相关文件归档以备将来使用。
* 在这一过程中, 我们通常是先对照合同内容和范围说明书等资料, 对卖方产品进行一一核对, 确认无误后, 再组织相关专家和干系人对采购件的质量、性能等关键指标进行评审。
  待所有工作完成后, 再依据合同里对本次采购的指南性条款结束采购。
  在采购结束后, 我们还对整个采购活动资料进行整理归纳并总结经验教训存入组织经验库。

# 经过我们团队历时 8 个月的辛苦努力, 本项目于 2024 年 5 月如期顺利完工并通过客户验收。
  总结整个项目的实施, 我认为本项目的成功得益于我采购管理的成功运用。
  通过科学的规划并制订了一份切实可行的采购计划, 使得项目采购活动能有条不紊的进行; 通过招标、谈判等多种采购方法的实施, 使得项目总是能以最优的价格获得最好的产品或服务; 通过持续监控采购活动, 使得项目采购件总是能保质保量按时交付。
  通过这一系列方法的实施, 建立了一套高效的采购管理机制, 为项目目标的成功实现提供了有效保障。
* 当然还是有不足之处, 比如因过于压缩团建活动经费, 导致团建活动效果并不理想。经征求总经理意见并获得批准后, 我将剩余部分经费以伙食补贴的形式发放给了大家, 大家对此都非常满意。
  在今后的工作中我将更加注重这方面的管理, 不断的学习充电, 努力提升自身管理水平, 为我国的信息化建设贡献自己的力量。
```
