# 文章_项目管理_XX市XX区人民医院HRP信息系统建设项目(人力资源管理)实例_GF_2024-04-09

本文将通过 XX市XX区人民医院HRP信息系统建设项目实例 来论述 `人力资源管理` 在其中的作用。

## 人力资源管理定义

**信息系统项目管理师教材定义**:

```txt
项目人力资源管理的目的是根据项目需要规划并组建项目团队，对团队进行有效的指导和管理，以保证他们可以完成项目任务，实现项目目标。
```

## 如何进行项目人力资源管理

**(1) 规划人力资源管理**:

```txt
规划人力资源管理:
规划人力资源管理是识别和记录项目角色、职责、所需技能、报告关系、并编制人员配备管理计划的过程。

规划人力资源管理的技术:
组织图和职位描述、组织理论、专家判断等

规划人力资源管理的输出:
人力资源管理计划
```

**(2) 组建项目团队**:

```txt
组建项目团队:
组建项目团队是确认人力资源的可用情况，并为开展项目活动而组建团队的过程。
本过程的主要收益是: 指导团队选择和职责分配，组建一个成功的团队。

组建项目团队的方法:
预分派、谈判、招募、虚拟团队等

组建项目团队的输出:
项目人员分派、资源日历等
```

**(3) 建设项目团队**:

```txt
建设项目团队:
建设项目团队是提高工作能力，促进团队成员互动，改善团队整体氛围，以提高项目绩效的过程。
本过程的主要收益是: 改进团队协作，增强人际技能，激励团队成员，降低人员离职率，提升整体项目绩效。

建设项目团队的方法:
培训、团队建设活动、认可与奖励等

建设项目团队的输出:
团队绩效评价、事业环境因素更新
```

**(4) 管理项目团队**:

```txt
管理项目团队:
管理项目团队是跟踪团队成员工作表现，提供反馈，解决问题并管理团队变更，优化项目绩效的过程。
本过程的主要收益是: 影响团队行为，管理冲突，解决问题，并评估团队成员的绩效。

管理项目团队的技术:
观察和交谈、项目绩效评估、冲突管理等

管理项目团队的输出:
变更请求、项目管理计划更新等
```

## XX市XX区人民医院HRP信息系统建设项目(人力资源管理)实例

**项目背景**:

```txt
# 2023 年 9 月, 我公司通过公开招标的方式中标了 XX 市 XX 区人民医院 HRP 信息系统建设项目, 我作为乙方项目经理全程参与了该项目的建设工作。
  本项目合同金额 3,630,000.00 元, 主要建设内容包括基础平台系统、财务会计系统、全面预算管理系统、科室成本核算系统、项目成本核算系统、病种成本核算系统、固定资产管理系统、合同管理系统、人力资源管理系统、统一对账系统、智能发放系统、报销管理系统、移动应用、决策分析(BI)、接口服务的建设。
  建设工期 8 个月, 前 5 个月内完成项目实施建设, 后 2 个月上线试运行, 最终验收 1 个月。
  该项目采用 B/S 架构, 前端展现包括系统入口、总览和数据仪表盘、小程序和移动APP, 后端主要采用Java、C#等语言开发模块, 实现预算编制模块、成本控制模块、资源管理模块、财务分析模块、凭证管理模块、成本核算模块、员工信息模块、薪酬绩效模块、报表与分析模块、集成模块等。
  关键技术指标要求实现系统的用户、权限、引入、输出、日志等功能, 要求支持 EAI 集成, 要求实现 PDM 系统接口、配置、规则等, 要求实现财务会计和预算会计双重功能平行记账、支持新会计制度要求的相关会计报表的生成, 要求实现资产条码管理, 报销单自动分摊、归集等。
  该项目于 2024 年 5 月顺利完工并通过客户验收, 获得了客户的肯定。
  本文以该项目为例, 结合作者实践经验, 探讨信息化系统项目建设过程中在人力资源管理方面经常遇到的问题及解决方法。
  主要就如何做好人力资源管理规划、组建项目团队、建设项目团队、管理项目团队这四个方面来进行阐述。最后总结项目中的成功经验, 以及存在的不足之处和具体改进措施。
```

**人力资源管理论述**:

```txt
# 为了更好的相应投标要求，我公司在投标前就已经对整个系统的解决方案和人员架构做了概要规划，拟派出的项目管理团队包括项目经理、项目助理、技术负责人各 1 名，开发组长 2 名，测试组长 1 名。
* 合同签订后，我们管理团队对整个项目进行了进一步的规划，其中最主要的当属人员配备方面的计划。
  怎样对团队结构进行合理的优化使之既能发挥整体协作优势又能适当降低成本; 怎样建设团队以提高项目绩效; 怎样管理团队以优化项目绩效，这些都是本项目需要重点考虑的问题。
  因此我认为有效的人力资源管理是本项目成功的关键。

# 鉴于人力资源管理在本项目中的重要性, 作为项目经理我在本项目中除了对其它各管理领域恪尽职守外, 还特别对人力资源管理从以下几个方面进行了管理:

# 一、规划人力资源管理
* 规划人力资源管理是识别和记录项目角色、职责、所需技能以及报告关系，并编制人员配备管理计划的过程。
  在规划人力资源管理时，势必要对人工费率和相关奖励方案等进行一个合理的优化，以确保项目成本最优。
  由于项目中的所有活动都是由人来完成，在进行成本管理时，除了直接的人工成本外，人员的绩效也直接决定着项目成本的高低。
  人员经验越丰富、技能水平越高那么团队容易更高效，但是人工费用也越高;
  奖励越丰厚团队成员就越积极性，但是成本支出也越高，所以必须要对团队成员结构进行一个合理的搭配，使得团队既能满足项目需要又能降低人工成本。
* 在本项目中，公司领导与我们开会讨论后一致认为，相似类型的项目我们公司已经有不少经验积累。
  在这个项目中，我们只需要把握好团队核心成员，然后 "以老带新"。
  这样既能节约项目成本，又能为公司培养人才，也利于公司长远发展。
  于是决定将公司最近校招的新成员，以 1 个老员工带 1 个新员工的比例安排到本项目组。

# 二、组建项目团队
* 在规划完人力资源管理后，我开始组建项目团队。
  由于团队核心成员已分派好，且公司安排了不少新员工，剩下的部分岗位人员我们很快就招募齐全。

# 三、建设项目团队
* 建设项目团队就是要提高工作能力，促进团队成员互动，改善团队整体氛围，以提高项目绩效。
* 我们项目组成员普遍比较年轻，活泼、肯干、肯冲是年轻人的性格特点。
  为了有效融合整个团队增强团队凝聚力，我喜欢在上班时请大家喝上几杯奶茶，饭点后和大家来到公司娱乐室打上几杆台球。
  一切团建活动让年轻人做主，他们喜欢打篮球，我就来个篮球赛。
  他们新来咋到，想出去看看这个城市的风景，我就组织大家去市内比较知名的景点游玩。
  他们在工作中常会提出比较新颖的思路和方法，而敢于发言，这个我们进行肯定并适当引导。
  光有氛围还不够，我们还有一系列的激励措施与培训计划，要让他们能够觉得能够学到东西、看到希望，要让他们与我们的团队一起成长。

# 四、管理项目团队
* 管理项目团队是跟踪团队成员工作表现，提供反馈，解决问题并管理团队变更，以优化项目绩效的过程。
* 在系统开发与实施阶段，我通过监控项目状态发现近期开发二组的效率比较低下。
  进度已经出现滞后，并且成本也发生了超支。经与小组长沟通交流后得知，是 1 名老员工最近因为家里事情比较多，请假有点频繁，而令 1 名老员工最近比较闹情绪。
  受他们的影响，他们带领的 2 名新员工近期也状态低迷。
  了解到问题原因后，我一方面与请假的那名员工取得了联系。
  就他的近况进行了沟通，与他确定了具体上岗日期。
  另一方面，我与闹情绪的老员工进行了深入沟通，其认为自己任务多，现在他还要带 2 名新员工，钱少事多身心疲惫。
  考虑到其状态不稳定，后期可能还会出现突发事件。
  于是我与技术负责人商量后决定，将这名员工暂且调到一组。
  从别的项目组借调 2 名熟练工程师过来，然后要求小组长加强对新员工的技能指导与培训，使他们尽快融入到工作中来，最终通过此次变更的实施使得项目进度与成本均恢复正常。

# 经过我们团队历时 8 个月的辛苦努力, 本项目于 2024 年 5 月如期顺利完工并通过客户验收。
  总结整个项目的实施, 我认为本项目的成功得益于人力资源和成本管理的有效运用。
  通过科学的配备团队成员结构，使得团队既能有效合作又能降低人力成本; 通过采用适合我们团队的建设方法来建设团队，使得团队得以高效运作; 通过及时跟踪与发现团队成员状态，解决了团队问题，优化了项目绩效。
  通过这一系列方法的实施, 建立了一套高效的人力资源管理机制, 充分发挥了项目中人的作用，为项目的成功实现提供了有效保障。
* 当然还是有不足之处, 比如因过于压缩团建活动经费，导致团建活动并不理想。
  经征求总经理一件并获得批准后，我将剩余活动经费以伙食补贴的形式发放给了大家，大家对此都非常满意。
  在今后的工作中我将更加注重团队建设工作, 不断的学习充电, 努力提升自身管理水平, 为我国的信息化建设贡献自己的力量。
```
