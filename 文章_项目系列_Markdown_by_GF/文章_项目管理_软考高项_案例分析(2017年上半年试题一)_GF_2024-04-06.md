# 文章_项目管理_软考高项_案例分析(2017年上半年试题一)_GF_2024-04-06

## 信息系统项目管理师(高级) 案例分析 (2017年上半年试题一)

某项目工期为 6 个月, 该项目的项目经理在第 3 个月末对项目进行了中期检查, 检查结果表明完成了计划进度的 90%, 相关情况见下表 (单位: 万元), 表中活动存在 F-S 关系。

|序号|    活动|第1月|第2月|第3月|第4月|第5月|第6月|PV值|
|----|----    |---- |---- |---- |---- |---- |---- |----|
|   1|编制计划|    4|    4|     |     |     |     |   8|
|   2|需求调研|     |    6|    6|     |     |     |  12|
|   3|概要设计|     |     |    4|    4|     |     |   8|
|   4|数据设计|     |     |     |    8|    4|     |  12|
|   5|详细设计|     |     |     |     |    8|    2|  10|
|    |  月度PV|    4|   10|   10|   12|   12|    2|    |
|    |  月度AC|    4|   11|   11|     |     |     |    |

\[问题1\] (8分)<br>
计算中期检查时项目的 CPI, CV和SV, 以及 "概要设计" 活动的 EV 和 SPI。

\[问题2\] (4分)<br>
如果按照当前的绩效, 计算项目的 ETC 和 EAC。

\[问题3\] (8分)<br>
请对该项目目前的进展情况作出评价。如果公司规定, 在项目中期评审中, 项目的进度绩效指标和成本绩效指标在计划值的正负 10% 即为正常, 则该项目是否需要采取纠正措施? 如需要, 请说明可采取哪些纠正措施进行成本控制; 如不需要, 请说明理由。

\[问题4\] (5分)<br>
结合本案例, 判断下列选项的正误 (填写在答题纸的对应栏内, 正确的选项填写 "√", 错误的选项填写 "×"):<br>
(1)应急储备是包含在成本基准内的一部分预算, 用来应对已经接受的已识别风险, 并已经指定应急或减轻措施的已识别风险。(    )<br>
(2)管理储备主要应对项目的 "已知-未知" 风险, 是为了管理控制的目的而特别留出的项目预算。(    )<br>
(3)管理储备是项目成本基准的有机组成部分, 不需要高层管理者审批就可以使用。(    )<br>
(4)成本基准就是项目的总预算, 不需要按照项目工作分解结构和项目生命周期进行分解。(    )<br>
(5)成本管理过程及其使用的工具和技术会因应用领域的不同而变化, 一般在项目生命期定义过程中对此进行选择。(    )

## 解题思路

```txt
[问题1] (8分)
计算中期检查时项目的 CPI, CV和SV, 以及 "概要设计" 活动的 EV 和 SPI。

    * 注意: 在第 3 个月末对项目进行了中期检查, 先观察截止第 3 个月的项目情况。
    
    +----+--------+-----+-----+-----+----+
    |序号|    活动|第1月|第2月|第3月|PV值|
    +----+--------+-----+-----+-----+----+
    |   1|编制计划|    4|    4|     |   8|
    |   2|需求调研|     |    6|    6|  12|
    |   3|概要设计|     |     |    4|   4|
    |   4|数据设计|     |     |     |    |
    |   5|详细设计|     |     |     |    |
    |    |  月度PV|    4|   10|   10|    |
    |    |  月度AC|    4|   11|   11|    |
    +----+--------+-----+-----+-----+----+
    
    截止第 3 个月, 计划完成 PV 值:
        PV = 4 + 4 + 6 + 6 + 4 = 24
    
    截止第 3 个月, 实际成本 AC 值:
        AC = 4 + 11 + 11 = 26 (实际产生了 "26万" 的成本)
    
    截止第 3 个月, 挣值 EV 值:
        EV = PV * 0.9 = 21.6 (实际完成了 "21.6万" 的工作)
    
    截止第 3 个月, 成本绩效指数 CPI 值:
        CPI = EV / AC = 21.6 / 26 = 0.83 (实际完成的值距离实际投入的成本还差 "17%")
    
    截止第 3 个月, 成本偏差 CV 值:
        CV = EV - AC = 21.6 - 26 = -4.4 (完成的值距离实际投入的成本还差 "4.4万")

    截止第 3 个月, 进度偏差 SV 值:
        SV = EV - PV = 21.6 - 24 = -2.4 (完成的值距离计划值还差 "2.4万")
    
    * 注意: 由于表中活动存在 F-S 关系 (Finish-Start), 即当前活动完成之后才能开始下一个活动, 例如: "编制计划" 在第 2 月完成后, 才能在第 2 月开始 "需求调研"。
    
    根据题干信息, 在开始 "需求调研" 时, 也就意味着 "编制计划" 已经完成, 在开始 "概要设计" 时, 也就意味着 "需求调研" 已经完成。
    
    由此可以推断, 在开始 "概要设计" 时, 已完成的 EV = 4 + 4 + 6 + 6 = 20
    
    而截止第 3 个月, 实际应该完成 PV = 4 + 4 + 6 + 6 + 4 = 24, 然而实际只完成了 EV = PV * 0.9 = 21.6, 也就是 "概要设计" 出现了延误, 只完成了 21.6 - 20 = 1.6
    
    因此,
    
    截止第 3 个月, "概要设计" 活动的 EV 值:
        EV = 21.6 - 20 = 1.6 ("概要设计" 实际完成了 "1.6万" 的工作)

    截止第 3 个月, "概要设计" 活动的 SPI 值:
        SPI = 1.6 / 4 = 0.4 ("概要设计" 完成的值只有计划值的 "40%")
    
    总结,
    
    中期检查时项目的 CPI 为 0.83, 项目的 CV 为 -4.4, 项目的 SV 为 -2.4, "概要设计" 活动的 EV 为 1.6, "概要设计" 活动的 SPI 为 0.4。

[问题3] (8分)
请对该项目目前的进展情况作出评价。如果公司规定, 在项目中期评审中, 项目的进度绩效指标和成本绩效指标在计划值的正负 10% 即为正常, 则该项目是否需要采取纠正措施? 如需要, 请说明可采取哪些纠正措施进行成本控制; 如不需要, 请说明理由。

    根据 [问题1] 计算结果, 截止第 3 个月, 该项目的成本绩效指数 CPI 值:
        CPI = EV / AC = 21.6 / 26 = 0.83 (实际完成的值距离实际投入的成本还差 "17%")
    
    截止第 3 个月, 该项目的进度绩效指数 SPI 值:
        SPI = EV / PV = 21.6 / 24 = 0.90 (实际完成的值距离计划值还差 "10%")
        
    总结,
    
    该项目目前的成本绩效指数 CPI 为 83%, 成本超支, 该项目目前的进度绩效指数 SPI 为 90%, 偏差度在 10% 以内, 需要采取措施。
    
    可采取的措施包括: 提高项目团队绩效, 给团队成员培训, 提高资源效率, 替换低效率资源。

[问题4] (5分)
结合本案例, 判断下列选项的正误 (填写在答题纸的对应栏内, 正确的选项填写 "√", 错误的选项填写 "×"):
(1)应急储备是包含在成本基准内的一部分预算, 用来应对已经接受的已识别风险, 并已经指定应急或减轻措施的已识别风险。(  √  )
(2)管理储备主要应对项目的 "已知-未知" 风险, 是为了管理控制的目的而特别留出的项目预算。(  ×  )
(3)管理储备是项目成本基准的有机组成部分, 不需要高层管理者审批就可以使用。(  ×  )
(4)成本基准就是项目的总预算, 不需要按照项目工作分解结构和项目生命周期进行分解。(  ×  )
(5)成本管理过程及其使用的工具和技术会因应用领域的不同而变化, 一般在项目生命期定义过程中对此进行选择。(  √  )

    涉及知识:
    
    制定预算 - 项目预算组成
    +--------+
    |合同总价|
    +--------+
        ↑
    +--------+----+
    |项目预算|利润|
    +--------+----+
        ↑
    +----------------------+--------+
    |成本基准(绩效衡量基准)|管理储备|
    +----------------------+--------+
        ↑
    +--------------+--------+
    |工作包成本估算|应急储备|
    +--------------+--------+
        ↑
    +------------+------------+
    |活动成本估算|活动应急储备|
    +------------+------------+
    
    制定预算 - 储备分析
    +--------+----------------------------------------------------------------------------------+
    |成本基准|"经批准" 的按时间安排的 "成本支出计划"。                                          |
    +--------+----------------------------------------------------------------------------------+
    |        |针对 "未知的未知事件"。                                                           |
    |管理储备|项目经理在使用或支出管理储备前, 可能需要获得批准。                                |
    |        |管理储备 "不是项目成本基准的一部分", 但包含在项目总预算中。                       |
    |        |纳入挣值计算。                                                                    |
    +--------+----------------------------------------------------------------------------------+
    |应急储备|针对 "已知的未知事件", 即风险登记册中的已知风险, "是项目范围和成本基准的一部分"。 |
    +--------+----------------------------------------------------------------------------------+
```
