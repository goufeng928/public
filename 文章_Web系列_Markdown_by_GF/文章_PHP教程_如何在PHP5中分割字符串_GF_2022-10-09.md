PHP (PHP: Hypertext Preprocessor) 即 "超文本预处理器", 是在服务器端执行的脚本语言, 尤其适用于Web开发并可嵌入HTML中。

PHP 语法学习了 C语言, 吸纳 Java 和 Perl 多个语言的特色发展出自己的特色语法, 并根据它们的长项持续改进提升自己, 例如 Java 的面向对象编程, 该语言当初创建的主要目标是让开发人员快速编写出优质的 Web 网站。

PHP 同时支持面向对象和面向过程的开发, 使用上非常灵活。

## 使用 PHP5 内置函数 explode() 分割字符串

**示例代码**:

```php
$str = "Hello,World";
$result = explode(",", $str);
```

**输出**:

```txt
Array ( [0] => Hello [1] => World )
```

## 使用 PHP5 正则表达式函数 preg_split() 分割字符串

**示例代码**:

```php
$str = "Hello,World";
$result = preg_split("/,/", $str);
```

**输出**:

```txt
Array ( [0] => Hello [1] => World )
```

## 使用 PHP5 的 substr() 和 strpos() 函数组合分割字符串

**示例代码**:

```php
$str = "Hello,World";
$delimiter = ",";
$length = strlen($delimiter);
$result = [];
while (($pos = strpos($str, $delimiter)) !== false) {
    $result[] = substr($str, 0, $pos);
    $str = substr($str, $pos + $length);
}
$result[] = $str;
```

**输出**:

```txt
Array ( [0] => Hello [1] => World )
```

## 使用 PHP5 的 strtok() 函数分割字符串

**示例代码**:

```php
$str = "Hello,World";
$delimiter = ",";
$result = [];
$token = strtok($str, $delimiter);
while ($token !== false) {
    $result[] = $token;
    $token = strtok($delimiter);
}
```

**输出**:

```txt
Array ( [0] => Hello [1] => World )
```
